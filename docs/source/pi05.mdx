# π₀.₅ (Pi05) 策略

π₀.₅ 是一个来自 Physical Intelligence 的**具备开放世界泛化能力的视觉-语言-动作（Vision-Language-Action）模型**。LeRobot 的实现改编自其开源仓库 [OpenPI](https://github.com/Physical-Intelligence/openpi)。

## 模型概述

π₀.₅ 代表了从 π₀ 的重要演进，由 [Physical Intelligence](https://www.physicalintelligence.company/blog/pi05) 开发，旨在解决机器人领域的一大挑战：**开放世界泛化（open-world generalization）**。虽然机器人可以在受控环境中完成令人印象深刻的复杂任务，但 π₀.₅ 被设计为能够泛化到训练期间从未见过的全新环境和情境。

### 泛化挑战

正如 Physical Intelligence 所解释的，根本挑战不在于敏捷性或灵巧性的任务执行能力，而在于**泛化**——即在新环境中使用新物体正确完成任务的能力。设想一个机器人在不同家庭中打扫卫生：每个家庭的物品和位置都不同。泛化必须在多个层面上进行：

- **物理层面**：理解如何拿起勺子（握柄）或盘子（边缘），即使是在杂乱环境中的未见物体
- **语义层面**：理解任务语义，知道衣服和鞋子该放哪里（洗衣篮，而不是床上），以及清洁溅洒物应使用什么工具
- **环境层面**：适应如家庭、杂货店、办公室和医院等"杂乱"的真实世界环境

### 异构数据协同训练

π₀.₅ 的突破性创新在于**异构数据源的协同训练（co-training on heterogeneous data）**。该模型从以下数据中学习：

1. **多模态网络数据**：图像描述、视觉问答、目标检测
2. **语言指令**：人类逐步指导机器人完成复杂任务
3. **子任务指令**：高级语义行为标签（例如，整理床铺时的"拿起枕头"）
4. **跨形态机器人数据**：来自不同机器人平台、具有不同能力的数据
5. **多环境数据**：部署在多个不同家庭中的静态机器人数据
6. **移动操作数据**：约 400 小时的移动机器人示教数据

这种多样化的训练混合创建了一个"课程"，使其能够同时在物理、视觉和语义层面实现泛化。

## 安装要求

1. 按照[安装指南](./installation)安装 LeRobot。
2. 安装 Pi0.5 依赖项，运行：

   ```bash
   pip install -e ".[pi]"
   ```

## 使用方法

要在 LeRobot 配置中使用 π₀.₅，将策略类型指定为：

```python
policy.type=pi05
```

## 训练

### 训练命令示例

以下是在自有数据集上微调基础 π₀.₅ 模型的完整训练命令：

```bash
lerobot-train \
    --dataset.repo_id=your_dataset \
    --policy.type=pi05 \
    --output_dir=./outputs/pi05_training \
    --job_name=pi05_training \
    --policy.repo_id=your_repo_id \
    --policy.pretrained_path=lerobot/pi05_base \
    --policy.compile_model=true \
    --policy.gradient_checkpointing=true \
    --wandb.enable=true \
    --policy.dtype=bfloat16 \
    --policy.freeze_vision_encoder=false \
    --policy.train_expert_only=false \
    --steps=3000 \
    --policy.device=cuda \
    --batch_size=32
```

### 关键训练参数

- **`--policy.compile_model=true`**：启用模型编译以加快训练速度
- **`--policy.gradient_checkpointing=true`**：在训练中显著减少显存使用
- **`--policy.dtype=bfloat16`**：使用混合精度训练以提高效率
- **`--batch_size=32`**：训练的批量大小，根据你的 GPU 显存调整
- **`--policy.pretrained_path=lerobot/pi05_base`**：要进行微调的 π₀.₅ 基础模型，可选参数：
  - [lerobot/pi05_base](https://huggingface.co/lerobot/pi05_base)
  - [lerobot/pi05_libero](https://huggingface.co/lerobot/pi05_libero)（针对 Libero 数据集专门训练）

### 训练参数说明

| 参数                    | 默认值    | 说明                             |
| ----------------------- | ------- | -------------------------------- |
| `freeze_vision_encoder` | `false` | 不冻结视觉编码器（vision encoder） |
| `train_expert_only`     | `false` | 不冻结 VLM，训练所有参数           |

**💡 提示**：设置 `train_expert_only=true` 会冻结 VLM，仅训练动作专家（action expert）和投影层，从而以更低的显存消耗进行微调。

如果数据集未使用 `quantiles` 转换，可以使用以下命令进行转换：

```bash
python src/lerobot/scripts/augment_dataset_quantile_stats.py \
    --repo-id=your_dataset \
```

或者使用以下归一化映射来训练 pi05：`--policy.normalization_mapping='{"ACTION": "MEAN_STD", "STATE": "MEAN_STD", "VISUAL": "IDENTITY"}'`

## 相对动作

默认情况下，π₀.₅ 预测绝对动作。你可以启用**相对动作**，使模型预测相对于当前机器人状态的偏移量。这可以提高某些配置下的训练稳定性。

要使用相对动作，首先通过 CLI 在相对空间下重新计算数据集统计信息：

```bash
lerobot-edit-dataset \
    --repo_id your_dataset \
    --operation.type recompute_stats \
    --operation.relative_action true \
    --operation.chunk_size 50 \
    --operation.relative_exclude_joints "['gripper']" \
    --push_to_hub true
```

或者在 Python 中等价实现：

```python
from lerobot.datasets import LeRobotDataset, recompute_stats

dataset = LeRobotDataset("your_dataset")
recompute_stats(dataset, relative_action=True, chunk_size=50, relative_exclude_joints=["gripper"])
dataset.push_to_hub()
```

`chunk_size` 应与你的策略的 `chunk_size` 一致（π₀.₅ 默认为 50）。`relative_exclude_joints` 列出应保持为绝对空间的关节名称（例如夹爪指令）。使用 `--push_to_hub true` 将更新后的统计信息上传到 Hub。

然后启用相对动作进行训练：

```bash
lerobot-train \
    --dataset.repo_id=your_dataset \
    --policy.type=pi05 \
    --policy.use_relative_actions=true \
    --policy.relative_exclude_joints='["gripper"]' \
    ...
```

## 性能结果

### Libero 基准测试结果

π₀.₅ 在 Libero 基准测试套件上展示了强大的性能。为了比较并测试其 LeRobot 实现，我们在 Libero 数据集上对 libero 基础模型额外进行了 6000 步微调，并将结果与 OpenPI 参考结果进行了比较。

| 基准测试              | LeRobot 实现 | OpenPI 参考 |
| -------------------- | ------------ | ----------- |
| **Libero Spatial**   | 97.0%        | 98.8%       |
| **Libero Object**    | 99.0%        | 98.2%       |
| **Libero Goal**      | 98.0%        | 98.0%       |
| **Libero 10**        | 96.0%        | 92.4%       |
| **平均**             | 97.5%        | 96.85%      |

这些结果展示了 π₀.₅ 在多样化机器人操作任务中的强大泛化能力。要复现这些结果，可以按照 [Libero](https://huggingface.co/docs/lerobot/libero) 部分的说明操作。
