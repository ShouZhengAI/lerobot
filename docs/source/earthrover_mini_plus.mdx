EarthRover Mini Plus 是一款完全开源的移动机器人，通过 Frodobots SDK 连接云端。这让你可以控制机器人并录制数据集，用于训练 AI 模型。

## 你需要准备的

### 硬件

- EarthRover Mini 机器人
- 装有 Python 3.12 或更新版本的电脑
- 互联网连接

### 设置 Frodobots SDK

机器人需要 [Frodobots SDK](https://github.com/frodobots-org/earth-rovers-sdk) 在你的电脑上运行。步骤如下：

1. 下载并安装 SDK：

```bash
git clone https://github.com/frodobots-org/earth-rovers-sdk.git
cd earth-rovers-sdk
pip install -r requirements.txt
```

2. 保存凭证：

将 .env 变量写入，包含 Frodobots 团队提供的 SDK API 密钥和机器人名称。

```bash
SDK_API_TOKEN=your_sdk_api_token_here
BOT_SLUG=your_bot_slug_here
CHROME_EXECUTABLE_PATH=/path/to/chrome_or_chromium
# 默认为 MAP_ZOOM_LEVEL=18 https://wiki.openstreetmap.org/wiki/Zoom_levels
MAP_ZOOM_LEVEL=18
MISSION_SLUG=your_mission_slug_here
# 图像质量，范围 0.1 到 1.0（默认：0.8）
# 建议：0.8 以获得更好的性能
IMAGE_QUALITY=0.8
# 图像格式：jpeg、png 或 webp（默认：png）
# 建议：jpeg 以获得更好的性能和更低的带宽使用
IMAGE_FORMAT=jpeg
```

3. 启动 SDK：

```bash
hypercorn main:app --reload
```

4. 打开网页浏览器访问 `http://localhost:8000`，然后点击"加入"

SDK 提供以下功能：

- 前置和后置摄像头的实时视频

> [!IMPORTANT]
> 必须先运行 SDK，然后才能使用机器人。

## 安装 LeRobot

请按照我们的[安装指南](./installation)安装 LeRobot。

在基础安装之外，还需安装带硬件依赖的 EarthRover Mini：

```bash
pip install -e ".[hardware]"
```

## 工作原理

机器人通过互联网进行通信：

- **移动指令**：通过 SDK 发送
- **摄像头视频**：从 SDK 接收
- **机器人信息**：电量、位置、速度等来自 SDK

你无需连接任何线缆——一切通过 SDK 工作。

## 校准

无需校准！只要 SDK 在运行，机器人即可直接使用。

## 控制机器人

你可以使用键盘控制机器人——就像用 WASD 键玩电子游戏一样。

### 键盘控制

| 键位 | 动作 |
| --- | --- |
| W | 向前移动 |
| S | 向后移动 |
| A | 左转（同时向前移动） |
| D | 右转（同时向前移动） |
| Q | 原地向左旋转 |
| E | 原地向右旋转 |
| X | 停止所有移动 |
| +/= | 增加速度 |
| - | 降低速度 |
| ESC | 断开连接 |

### 速度设置

你可以调节机器人的移动速度：

- **前进/后退速度**：默认全速（1.0）
- **转弯速度**：默认全速（1.0）
- **速度调整**：使用 +/- 键，每次调整 0.1

### 试试看

在录制数据前先试驾机器人：

```python
from lerobot.robots.earthrover_mini_plus import EarthRoverMiniPlus, EarthRoverMiniPlusConfig
from lerobot.teleoperators.keyboard import KeyboardRoverTeleop, KeyboardRoverTeleopConfig

# 初始化机器人
robot_config = EarthRoverMiniPlusConfig()
robot = EarthRoverMiniPlus(robot_config)

# 初始化遥操作器
teleop_config = KeyboardRoverTeleopConfig(
    linear_speed=1.0,
    angular_speed=1.0,
    speed_increment=0.1
)
teleop = KeyboardRoverTeleop(teleop_config)

# 连接
robot.connect()
teleop.connect()

# 遥操作（使用键盘控制）
try:
    while True:
        action = teleop.get_action()
        robot.send_action(action)
except KeyboardInterrupt:
    pass
finally:
    robot.disconnect()
    teleop.disconnect()
```

> [!TIP]
> 如果你使用的是 Mac，可能需要授予终端键盘访问权限才能进行遥操作。前往系统设置 > 安全性与隐私 > 输入监视，勾选终端应用。

## 录制数据

一旦你能熟练驾驶机器人，就可以开始录制数据来训练 AI 模型。系统会记录：

- **你的操作**：如何移动机器人（前进、后退、转弯）
- **机器人看到的内容**：
  - 两个摄像头的视频
  - 机器人速度和方向
  - 电量和位置
  - GPS 定位和信号
  - 其他传感器数据
- **发生时间**：所有数据的时间戳

### 设置 Hugging Face

我们使用 Hugging Face 在线存储你的数据。首先，使用 [Hugging Face 设置](https://huggingface.co/settings/tokens)页面中的令牌登录：

```bash
hf auth login --token ${HUGGINGFACE_TOKEN} --add-to-git-credential
```

保存你的 Hugging Face 用户名：

```bash
HF_USER=$(hf auth whoami | awk -F': *' 'NR==1 {print $2}')
echo $HF_USER
```

### 开始录制

使用标准的录制命令：

```bash
lerobot-record \
    --robot.type=earthrover_mini_plus \
    --teleop.type=keyboard_rover \
    --dataset.repo_id=你的用户名/数据集名称 \
    --dataset.num_episodes=2 \
    --dataset.fps=10 \
    --dataset.single_task="绕过障碍物导航" \
    --dataset.streaming_encoding=true \
    --dataset.encoder_threads=2 \
    # --dataset.rgb_encoder.vcodec=auto \
    --display_data=true
```

将 `你的用户名/数据集名称` 替换为你的 Hugging Face 用户名和数据集名称。

### 保存的内容

你的数据集包含以下内容：

**你的动作（2 个特征）**：

- `linear_velocity`：你向前/向后移动了多少
- `angular_velocity`：你向左/向右转了多少

**机器人观测值（24 个特征）**：

- 前置摄像头视频
- 后置摄像头视频
- 当前速度
- 电量
- 朝向
- GPS（经度、纬度、信号强度）
- 网络信号强度
- 振动等级
- 灯光状态（开/关）
- 加速度计（x、y、z）
- 陀螺仪（x、y、z）
- 磁力计（x、y、z）
- 车轮转速（4 个轮子）

### 数据存储位置

在你的电脑上：`~/.cache/huggingface/lerobot/{repo-id}`

录制完成后，数据会自动上传到你的 Hugging Face 页面：

```bash
echo https://huggingface.co/datasets/${HF_USER}/earthrover-navigation
```

你的数据集将被标记为 `LeRobot`，便于社区发现。
