以下步骤将说明如何组装我们的旗舰机器人 SO-101。

## 采购零件

请参考此 [README](https://github.com/TheRobotStudio/SO-ARM100)。其中包含物料清单（Bill of Materials）、零件采购链接以及 3D 打印零件的说明。
如果你是第一次打印或没有 3D 打印机，请参考相关建议。

## 安装 LeRobot 🤗

安装 LeRobot 请参考我们的 [安装指南](./installation)

除上述说明外，你还需要安装 Feetech SDK：

```bash
pip install -e ".[feetech]"
```

## 分步组装说明

从动臂（follower arm）使用 6 个 STS3215 电机，减速比为 1/345。而主臂（leader arm）使用了三种不同减速比的电机，以确保既能支撑自身重量，又无需费力即可移动。每个关节所需的电机如下表所示。

| 主臂关节           | 电机 | 减速比 |
| ----------------- | :--: | :----: |
| 基座/肩部旋转      |  1   | 1 / 191 |
| 肩部抬升           |  2   | 1 / 345 |
| 肘部弯曲           |  3   | 1 / 191 |
| 腕部弯曲           |  4   | 1 / 147 |
| 腕部旋转           |  5   | 1 / 147 |
| 夹爪               |  6   | 1 / 147 |

## 配置电机

### 1. 找到每只机械臂对应的 USB 端口

要找到每个总线舵机适配器（bus servo adapter）对应的端口，请通过 USB 连接 MotorBus 到电脑并通电。运行以下脚本，并在提示时断开 MotorBus：

```bash
lerobot-find-port
```

<hfoptions id="example">
<hfoption id="Mac">

示例输出：

```
Finding all available ports for the MotorBus.
['/dev/tty.usbmodem575E0032081', '/dev/tty.usbmodem575E0031751']
Remove the USB cable from your MotorsBus and press Enter when done.

[...断开对应的主臂或从动臂，按 Enter...]

The port of this MotorsBus is /dev/tty.usbmodem575E0032081
Reconnect the USB cable.
```

其中找到的端口为：`/dev/tty.usbmodem575E0032081`，对应你的主臂或从动臂。

</hfoption>
<hfoption id="Linux">

在 Linux 上，你可能需要运行以下命令授予 USB 端口访问权限：

```bash
sudo chmod 666 /dev/ttyACM0
sudo chmod 666 /dev/ttyACM1
```

示例输出：

```
Finding all available ports for the MotorBus.
['/dev/ttyACM0', '/dev/ttyACM1']
Remove the usb cable from your MotorsBus and press Enter when done.

[...断开对应的主臂或从动臂，按 Enter...]

The port of this MotorsBus is /dev/ttyACM1
Reconnect the USB cable.
```

其中找到的端口为：`/dev/ttyACM1`，对应你的主臂或从动臂。

</hfoption>
</hfoptions>

### 2. 设置电机 ID 和波特率

每个电机在总线上通过唯一的 ID 标识。新电机通常默认 ID 为 `1`。为了让电机和控制器之间正常通信，我们首先需要为每个电机设置不同的唯一 ID。此外，总线上数据的传输速度由波特率决定。为了能够相互通信，控制器和所有电机需要配置相同的波特率。

为此，我们首先需要用控制器逐个连接每个电机来设置这些参数。由于我们将把这些参数写入电机内部存储器的非易失性区域（EEPROM），因此只需执行一次即可。

如果你是从其他机器人上复用电机，也可能需要执行此步骤，因为 ID 和波特率很可能不匹配。

以下视频展示了设置电机 ID 的操作步骤。

#### 从动臂

将电脑的 USB 线和电源连接到从动臂的控制器板。然后，运行以下命令或使用上一步获得的端口运行 API 示例。你还需要通过 `id` 参数为从动臂命名。

<hfoptions id="setup_motors">
<hfoption id="命令">

```bash
lerobot-setup-motors \
    --robot.type=so101_follower \
    --robot.port=/dev/tty.usbmodem585A0076841  # <- 在此粘贴上一步找到的端口
```

</hfoption>
<hfoption id="API 示例">

<!-- prettier-ignore-start -->
```python
from lerobot.robots.so_follower import SO101Follower, SO101FollowerConfig

config = SO101FollowerConfig(
    port="/dev/tty.usbmodem585A0076841",
    id="my_awesome_follower_arm",
)
follower = SO101Follower(config)
follower.setup_motors()
```
<!-- prettier-ignore-end -->

</hfoption>
</hfoptions>

你应该会看到以下提示：

```bash
Connect the controller board to the 'gripper' motor only and press enter.
```

按照指示，仅插入夹爪电机。确保它是板上唯一连接的电机，且该电机尚未以菊花链方式连接到其他电机。按下 `[Enter]` 后，脚本将自动设置该电机的 ID 和波特率。

<details>
<summary>故障排除</summary>

如果此时出现错误，请检查线缆并确保它们连接正确：

<ul>
  <li>电源</li>
  <li>电脑与控制器板之间的 USB 线</li>
  <li>控制器板到电机的 3 针线缆</li>
</ul>

如果你使用的是 Waveshare 控制器板，请确保两个跳线设置在 `B` 通道（USB）。

</details>

然后你应该会看到以下消息：

```bash
'gripper' motor id set to 6
```

接着是下一步提示：

```bash
Connect the controller board to the 'wrist_roll' motor only and press enter.
```

你可以断开控制器板上的 3 针线缆，但另一端可以保留连接到夹爪电机，因为它已经安装到位。现在，将另一根 3 针线缆插入腕部旋转电机并连接到控制器板。和之前的电机一样，确保它是板上唯一连接的电机，且该电机本身没有连接到任何其他电机。

按照提示为每个电机重复此操作。

> [!TIP]
> 在按 Enter 之前，请在每个步骤检查你的线缆连接。例如，电源线可能在操作板子的过程中松脱。

完成后，脚本将正常结束，此时电机即可使用。现在，你可以将每个电机的 3 针线缆依次连接到下一个电机，并将第一个电机（ID=1 的'肩部旋转'电机）的线缆连接到控制器板，该控制器板现在可以固定在机械臂基座上。

### 清洁零件

清除所有 3D 打印零件上的支撑材料。最简单的方法是使用小螺丝刀从支撑材料下方将其撬起。

建议在继续组装之前，先在电机上安装一根 3 针线缆。

### 关节 1

- 安装两个电机舵盘（motor horn）。用 M3x6mm 螺丝固定上方舵盘。下方舵盘无需螺丝。
- 将第一个电机放入基座。
- 用 4 颗 M2x6mm 螺丝（最小号螺丝）固定电机。顶部 2 颗，底部 2 颗。
- 套上第一个电机固定座，用 2 颗 M2x6mm 螺丝固定（每侧 1 颗）。
- 安装肩部部件。
- 用顶部 4 颗 M3x6mm 螺丝和底部 4 颗 M3x6mm 螺丝拧紧肩部部件。
- 安装肩部电机固定座。

## 校准

接下来，你需要校准机器人，以确保主臂和从动臂在相同物理位置时具有相同的位置值。
校准过程非常重要，因为它可以使在一个机器人上训练的神经网络也能在另一个机器人上工作。

#### 从动臂

运行以下命令或 API 示例来校准从动臂：

<hfoptions id="calibrate_follower">
<hfoption id="命令">

```bash
lerobot-calibrate \
    --robot.type=so101_follower \
    --robot.port=/dev/tty.usbmodem58760431551 \ # <- 你的机器人端口
    --robot.id=my_awesome_follower_arm # <- 为机器人指定唯一名称
```

</hfoption>
<hfoption id="API 示例">

<!-- prettier-ignore-start -->
```python
from lerobot.robots.so_follower import SO101FollowerConfig, SO101Follower

config = SO101FollowerConfig(
    port="/dev/tty.usbmodem585A0076891",
    id="my_awesome_follower_arm",
)

follower = SO101Follower(config)
follower.connect(calibrate=False)
follower.calibrate()
follower.disconnect()
```
<!-- prettier-ignore-end -->

</hfoption>
</hfoptions>

以下视频展示了如何进行校准。首先你需要将机器人移动到所有关节处于行程范围中间的位置。然后按 Enter 后，你需要让每个关节走完其完整的运动范围。
