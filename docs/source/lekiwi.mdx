下面我们将逐步说明如何组装 LeKiwi 移动机器人。

## 采购零件

请参考这份 [README](https://github.com/SIGRobotics-UIUC/LeKiwi)，其中包含物料清单（BOM）、零件采购链接以及 3D 打印零件的说明。如果是首次进行 3D 打印或者没有 3D 打印机，请注意查看相关建议。

### 有线版本

如果你使用的是**有线版** LeKiwi，可以跳过 Raspberry Pi 安装和 SSH 设置。在笔记本电脑和 LeKiwi 脚本以及用于遥操作的主臂脚本上，你都可以直接在笔记本电脑上运行所有命令。

## 在树莓派上安装软件

现在我们需要设置将在 LeKiwi 机器人上运行的远程电脑。通常是 Raspberry Pi，但也可以是任何能支持 5V 供电且有足够 USB 端口（2 个或以上）用于连接摄像头和电机控制板的电脑。

### 安装操作系统

关于 Raspberry Pi 及其 SD 卡的设置，请参考：[树莓派设置](https://www.raspberrypi.com/documentation/computers/getting-started.html)。其中说明了如何下载 [Imager](https://www.raspberrypi.com/software/) 来安装 Raspberry Pi OS 或 Ubuntu。

### 设置 SSH

完成树莓派设置后，应启用并配置 [SSH](https://www.raspberrypi.com/news/coding-on-raspberry-pi-remotely-with-visual-studio-code/)（安全外壳协议），这样你就可以从笔记本电脑登录到树莓派，无需在树莓派上连接屏幕、键盘和鼠标。关于如何进行设置，[这里](https://www.raspberrypi.com/documentation/computers/remote-access.html#ssh)有一份很好的教程。你可以在命令提示符（cmd）中登录树莓派，如果使用 VSCode，也可以使用[这个](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-ssh)扩展。

### 在树莓派上安装 LeRobot 🤗

在树莓派上，按照我们的[安装指南](./installation)安装 LeRobot。

此外，你还需要在树莓派上安装 Feetech SDK 和 ZeroMQ：

```bash
pip install -e ".[lekiwi]"
```

## 本地安装 LeRobot

如果你已在笔记本电脑/电脑上安装了 LeRobot，可以跳过此步骤；否则，请按照我们之前在树莓派上的步骤进行操作。

请参考我们的[安装指南](./installation)

此外，你还需要在笔记本电脑/电脑上安装 Feetech SDK 和 ZeroMQ：

```bash
pip install -e ".[lekiwi]"
```

太好了 :hugs:！LeRobot 安装已完成，现在我们可以开始组装 SO100/SO101 机械臂和移动底座了 :robot:。
之后每次使用 LeRobot 时，只需进入安装 LeRobot 的 `~/lerobot` 文件夹并运行相应命令即可。

### 配置电机

电机配置说明请参见 SO101 [文档](./so101#configure-the-motors)。除了设置机械臂电机的 ID，我们还需要设置移动底座电机的 ID。这些 ID 必须按特定顺序设置才能正常工作。下面是移动底座的电机 ID 和电机安装位置示意图。请注意，在 LeKiwi 上我们只使用一个电机控制板，这意味着轮子的电机 ID 为 7、8 和 9。

你可以运行以下命令为 LeKiwi 设置电机。它将先设置机械臂电机（ID 6 到 1），然后设置轮子电机（9、8、7）。

```bash
lerobot-setup-motors \
    --robot.type=lekiwi \
    --robot.port=/dev/tty.usbmodem58760431551 # <- 在此粘贴上一步找到的端口
```

### 通信故障排查

如果连接 Mobile SO100 时遇到问题，请按照以下步骤诊断和解决。

#### 1. 验证 IP 地址配置

确保命令或代码中使用了正确的树莓派 IP 地址。要查看树莓派的 IP 地址，请在树莓派命令行中运行：

```bash
hostname -I
```

#### 2. 检查笔记本电脑是否能访问树莓派

尝试从笔记本电脑 ping 树莓派：

```bash
ping <你的树莓派IP地址>
```

如果 ping 失败：

- 确保树莓派已开机并连接到同一网络。
- 检查树莓派上是否已启用 SSH。

#### 3. 尝试 SSH 连接

如果无法通过 SSH 连接到树莓派，可能是连接配置有误。请使用：

```bash
ssh <你的树莓派用户名>@<你的树莓派IP地址>
```

如果出现连接错误：

- 确保树莓派上已启用 SSH，运行以下命令：
  ```bash
  sudo raspi-config
  ```
  然后导航到：**Interfacing Options -> SSH** 并启用它。

### 标定

现在我们需要标定主臂和从臂。轮子电机不需要标定。
标定过程非常重要，因为它能使在一个机器人上训练的神经网络在另一个机器人上工作。

### 标定从臂（安装在移动底座上）

确保机械臂已连接到树莓派，然后运行以下脚本或 API 示例（通过 SSH 在树莓派上运行）来启动从臂标定：

```bash
lerobot-calibrate \
    --robot.type=lekiwi \
    --robot.id=my_awesome_kiwi # <- 为机器人指定一个唯一名称
```

我们统一了大多数机器人的标定方法，因此该 SO100 机械臂的标定步骤与 Koch 和 SO101 相同。首先，将机器人移动到每个关节处于其量程中间的位置，然后按 `Enter`。其次，让所有关节完成全范围运动。可以参考[此视频](https://huggingface.co/docs/lerobot/en/so101#calibration-video)了解 SO101 的相同标定过程。

### 有线版本

如果你使用的是**有线版** LeKiwi，请在笔记本电脑上运行所有命令。

### 标定主臂

然后，标定连接到笔记本电脑/电脑的主臂。在你的笔记本电脑上运行以下命令或 API 示例：

## 遥操作 LeKiwi

> [!TIP]
> 如果你使用的是 Mac，可能需要为 Terminal 授予键盘访问权限以进行遥操作。请前往系统偏好设置 > 安全性与隐私 > 输入监视，勾选 Terminal。

要进行遥操作，请通过 SSH 登录到树莓派，运行 `conda activate lerobot` 并执行以下命令：

```bash
python -m lerobot.robots.lekiwi.lekiwi_host --robot.id=my_awesome_kiwi
```

然后在你的笔记本电脑上，也运行 `conda activate lerobot` 并执行 API 示例，确保在 `examples/lekiwi/teleoperate.py` 中设置了正确的 `remote_ip` 和 `port`。

```bash
python examples/lekiwi/teleoperate.py
```

你的笔记本电脑上应该会显示类似这样的信息：`[INFO] Connected to remote robot at tcp://172.17.133.91:5555 and video stream at tcp://172.17.133.91:5556.` 现在你可以移动主臂，并使用键盘（w、a、s、d）控制前进、左移、后退、右移。使用（z、x）控制左转或右转。使用（r、f）来增加或减少移动机器人的速度。共有三种速度模式，如下表所示：

| 速度模式 | 线速度 (m/s) | 旋转速度 (deg/s) |
| -------- | ------------- | ---------------- |
| 快速     | 0.4           | 90               |
| 中速     | 0.25          | 60               |
| 慢速     | 0.1           | 30               |

| 按键 | 动作     |
| ---- | -------- |
| W    | 前进     |
| A    | 左移     |
| S    | 后退     |
| D    | 右移     |
| Z    | 左转     |
| X    | 右转     |
| R    | 加速     |
| F    | 减速     |

> [!TIP]
> 如果你使用不同的键盘，可以在 [`LeKiwiClientConfig`](https://github.com/huggingface/lerobot/blob/main/src/lerobot/robots/lekiwi/config_lekiwi.py) 中修改每个命令对应的按键。

## 录制数据集

熟悉遥操作后，你可以录制第一个数据集了。

我们使用 Hugging Face Hub 功能来上传数据集。如果之前未使用过 Hub，请确保可以通过 CLI 使用具有写入权限的令牌（token）登录，该令牌可以在 [Hugging Face 设置](https://huggingface.co/settings/tokens)中生成。

运行以下命令将令牌添加到 CLI：

```bash
hf auth login --token ${HUGGINGFACE_TOKEN} --add-to-git-credential
```

然后将 Hugging Face 仓库名称存入变量：

```bash
HF_USER=$(hf auth whoami | awk -F': *' 'NR==1 {print $2}')
echo $HF_USER
```

现在可以录制数据集了。要录制片段（episode）并将数据集上传到 Hub，请执行以下针对 LeKiwi 定制的 API 示例。请先在脚本中修改 `remote_ip`、`repo_id`、`port` 和 `task`。如果需要更长时间运行脚本，可以增加 `NB_CYCLES_CLIENT_CONNECTION`。

```bash
python examples/lekiwi/record.py
```

#### 数据集上传

数据集本地存储在 `~/.cache/huggingface/lerobot/{repo-id}` 文件夹中。数据录制完成后，数据集将上传到你的 Hugging Face 页面（例如 https://huggingface.co/datasets/cadene/so101_test），可通过以下命令查看：

```bash
echo https://huggingface.co/datasets/${HF_USER}/so101_test
```

你的数据集将自动标记为 `LeRobot`，方便社区查找，你也可以添加自定义标签（例如本例中的 `tutorial`）。

你可以在 Hub 上搜索 `LeRobot` [标签](https://huggingface.co/datasets?other=LeRobot)来查找其他 LeRobot 数据集。

#### 数据采集技巧

熟悉数据录制后，你可以创建用于训练的更大规模数据集。一个好的入门任务是：在不同位置抓取一个物体并将其放入收纳箱中。我们建议至少录制 50 个片段，每个位置约 10 个片段。保持摄像头固定，并在录制过程中保持一致的抓取行为。同时确保操作的物体在摄像头画面中可见。一个好的经验法则是：仅通过查看摄像头图像，你应该就能自己完成该任务。

在接下来的章节中，你将训练神经网络。在实现可靠的抓取性能后，你可以在数据采集过程中引入更多变化，例如更多的抓取位置、不同的抓取技巧以及改变摄像头位置。

避免过快引入过多变化，否则可能会影响结果。

如果想深入了解这一重要主题，可以查看我们撰写的关于"什么构成了好的数据集"的[博客文章](https://huggingface.co/blog/lerobot-datasets#what-makes-a-good-dataset)。

#### 故障排除

- 在 Linux 上，录制控制流程按键（方向键、Escape）可在 X11、Wayland 和 headless/SSH 会话中使用，只要你在交互式终端中运行录制（保持其处于焦点状态）——无需设置 `$DISPLAY`；字母等效键 `n` / `r` / `q` 也可以使用。请注意，**LeKiwi 底盘的键盘遥操作**有所不同：它依赖于全局按键后端，因此只能在 X11 会话、Windows 桌面或已授予辅助功能/输入监视权限的 macOS 上工作——不能在 Wayland 或无头机器上使用。详见 [pynput 限制说明](https://pynput.readthedocs.io/en/latest/limitations.html#linux)。
