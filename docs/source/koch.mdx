# Koch v1.1

在下面的步骤中，我们将说明如何组装 Koch v1.1 机器人。

## 订购并组装零件

请遵循此 [README](https://github.com/jess-moss/koch-v1-1) 中提供的零件采购和组装说明。这将指导你完成从端和主端机械臂的设置，如下图所示。

如需装配过程的可视化演示，可以参考[此视频教程](https://youtu.be/8nQIg9BwwTk)。

> [!WARNING]
> 自该视频制作以来，我们简化了配置阶段。因此，以下两点与视频中的说明不同：
>
> - 不要立即插入所有电机线缆，等待在[配置电机](#configure-the-motors)中收到指示后再操作。
> - 不要立即将控制板（PCB）拧到基座上，等待在[配置电机](#configure-the-motors)中收到指示后再操作。

## 安装 LeRobot 🤗

要安装 LeRobot，请遵循我们的[安装指南](./installation)

```bash
pip install -e ".[dynamixel]"
```

## 配置电机

### 1. 找到每条机械臂对应的 USB 端口

要找到每个总线舵机适配器的端口，请运行以下脚本：

```bash
lerobot-find-port
```

<hfoptions id="example">
<hfoption id="Mac">

示例输出：

```
Finding all available ports for the MotorBus.
['/dev/tty.usbmodem575E0032081', '/dev/tty.usbmodem575E0031751']
Remove the USB cable from your MotorsBus and press Enter when done.

[...断开对应的主端或从端机械臂并按下 Enter...]

The port of this MotorsBus is /dev/tty.usbmodem575E0032081
Reconnect the USB cable.
```

其中找到的端口是：`/dev/tty.usbmodem575E0032081`，对应你的主端或从端机械臂。

</hfoption>


示例输出：

```
[...断开对应的主端或从端机械臂并按下 Enter...]

The port of this MotorsBus is /dev/ttyACM1
Reconnect the USB cable.
```

其中找到的端口是：`/dev/ttyACM1`，对应你的主端或从端机械臂。

### 2. 设置电机 ID 和波特率

每个电机在总线上由一个唯一的 ID 标识。全新时，电机通常以默认 ID `1` 发货。为了使电机和控制器之间的通信正常工作，我们首先需要为每个电机设置不同且唯一的 ID。此外，总线上数据传输的速度由波特率决定。为了相互通信，控制器和所有电机都需要配置相同的波特率。

为此，我们首先需要将每个电机单独连接到控制器来设置这些参数。由于我们将把这些参数写入电机内部存储器的非易失性区域（EEPROM），因此只需执行此操作一次。

如果你是从其他机器人上重新使用电机，很可能也需要执行此步骤，因为 ID 和波特率很可能不匹配。

#### 从端（Follower）

将计算机的 USB 线和 5V 电源连接到从端机械臂的控制板。然后，运行以下命令或使用上一步获取的端口运行 API 示例。你还需要使用 `id` 参数为你的从端机械臂命名。

关于如何设置电机 ID 的视觉参考，请参阅[此视频](https://huggingface.co/docs/lerobot/en/so101#setup-motors-video)，其中我们展示了 SO101 机械臂的设置过程。

<hfoptions id="setup_motors">
<hfoption id="Command">

```bash
lerobot-setup-motors \
    --robot.type=koch_follower \
    --robot.port=/dev/tty.usbmodem575E0031751 # <- 在此粘贴上一步找到的端口
```

</hfoption>
<hfoption id="API example">

<!-- prettier-ignore-start -->
```python
from lerobot.robots.koch_follower import KochFollower, KochFollowerConfig

config = KochFollowerConfig(
    port="/dev/tty.usbmodem575E0031751",
    id="my_awesome_follower_arm",
)
follower = KochFollower(config)
follower.setup_motors()
```
<!-- prettier-ignore-end -->

</hfoption>
</hfoptions>

你会看到以下提示：

```
Connect the controller board to the 'gripper' motor only and press enter.
```

按照指示，插入夹爪电机。确保它是唯一连接到控制板的电机，并且该电机本身尚未通过菊花链连接到任何其他电机。当你按下 `[Enter]` 后，脚本将自动设置该电机的 ID 和波特率。

<details>
<summary>故障排除</summary>

如果此时出现错误，请检查线缆并确保它们已正确插入：

<ul>
  <li>电源</li>
  <li>计算机和控制板之间的 USB 线</li>
  <li>从控制板到电机的 3 针线缆</li>
</ul>

如果你使用的是 Waveshare 控制板，请确保两个跳线设置在 `B` 通道（USB）上。

</details>

然后你会看到以下信息：

```
'gripper' motor id set to 6
```

紧接着是下一条指示：

```
Connect the controller board to the 'wrist_roll' motor only and press enter.
```

你可以从控制板上断开 3 针线缆，但另一端可以保持连接在夹爪电机上，因为它已经在正确的位置。现在，将另一根 3 针线缆插入手腕滚动（wrist_roll）电机，并将其连接到控制板。与上一个电机一样，确保它是唯一连接到控制板的电机，且该电机本身未连接到任何其他电机。

按照指示为每个电机重复此操作。

> [!TIP]
> 在按 Enter 前检查每一步的接线。例如，当你操作控制板时，电源线可能会断开。

完成后，脚本将结束运行，此时电机已准备好使用。现在，你可以将每个电机的 3 针线缆连接到下一个电机，并将第一个电机（ID=1 的 shoulder pan 电机）的线缆连接到控制板，控制板现在可以安装到机械臂的基座上。

#### 主端（Leader）

对主端机械臂执行相同步骤，但相应地修改命令或脚本。


## 标定（Calibrate）

接下来，你需要标定机器人，以确保主端和从端机械臂在处于相同物理位置时具有相同的位置值。
标定过程非常重要，因为它可以使在一个机器人上训练好的神经网络在另一个机器人上正常工作。

#### 从端（Follower）

运行以下命令或 API 示例来标定从端机械臂：

<hfoptions id="calibrate_follower">
<hfoption id="Command">

```bash
lerobot-calibrate \
    --robot.type=koch_follower \
    --robot.port=/dev/tty.usbmodem58760431551 \ # <- 你的机器人端口
    --robot.id=my_awesome_follower_arm  # <- 为机器人指定唯一名称
```

</hfoption>
<hfoption id="API example">

<!-- prettier-ignore-start -->
```python
from lerobot.robots.koch_follower import KochFollowerConfig, KochFollower

config = KochFollowerConfig(
    port="/dev/tty.usbmodem585A0076891",
    id="my_awesome_follower_arm",
)

follower = KochFollower(config)
follower.connect(calibrate=False)
follower.calibrate()
follower.disconnect()
```
<!-- prettier-ignore-end -->

</hfoption>
</hfoptions>

我们统一了大部分机器人的标定方法。因此，此 Koch 机械臂的标定步骤与 SO100 和 SO101 的步骤相同。首先，我们需要将机器人移动到每个关节位于其运动范围中间的位置，然后按下 `Enter`。其次，我们移动所有关节遍历其完整的运动范围。SO101 相同过程的视频参考见[此处](https://huggingface.co/docs/lerobot/en/so101#calibration-video)。

#### 主端（Leader）

对标定主端机械臂执行相同的步骤，运行以下命令或 API 示例：

<hfoptions id="calibrate_leader">
<hfoption id="Command">

```bash
lerobot-calibrate \
    --teleop.type=koch_leader \
    --teleop.port=/dev/tty.usbmodem58760431551 \ # <- 你的机器人端口
    --teleop.id=my_awesome_leader_arm  # <- 为机器人指定唯一名称
```

</hfoption>
<hfoption id="API example">

<!-- prettier-ignore-start -->
```python
from lerobot.teleoperators.koch_leader import KochLeaderConfig, KochLeader

config = KochLeaderConfig(
    port="/dev/tty.usbmodem575E0031751",
    id="my_awesome_leader_arm",
)

leader = KochLeader(config)
leader.connect(calibrate=False)
leader.calibrate()
leader.disconnect()
```

</hfoption>
