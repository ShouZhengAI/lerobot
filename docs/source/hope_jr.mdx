# HopeJR

## 前置条件

- [硬件设置](https://github.com/TheRobotStudio/HOPEJr)

## 安装 LeRobot

按照[安装说明](https://github.com/huggingface/lerobot#installation)安装 LeRobot。

安装带有 HopeJR 依赖的 LeRobot：

```bash
pip install -e ".[hopejr]"
```

## 设备配置

在开始校准和操作之前，你需要识别每个 HopeJR 组件的 USB 端口。运行以下脚本来查找机械臂、灵巧手、数据手套和外骨骼的 USB 端口：

```bash
lerobot-find-port
```

该命令将显示可用的 USB 端口及其关联的设备。请记下端口路径（例如 `/dev/tty.usbmodem58760433331`、`/dev/tty.usbmodem11301`），因为在记录数据、回放片段或运行遥操作脚本时，你需要在 `--robot.port` 和 `--teleop.port` 参数中指定它们。

## 第 1 步：校准

在执行遥操作之前，需要对 HopeJR 的肢体进行校准。校准文件将保存在 `~/.cache/huggingface/lerobot/calibration` 目录下。

### 1.1 校准机器人灵巧手

```bash
lerobot-calibrate \
    --robot.type=hope_jr_hand \
    --robot.port=/dev/tty.usbmodem58760432281 \
    --robot.id=blue \
    --robot.side=right
```

运行校准脚本时，会弹出校准图形界面。手指关节的命名如下：

**拇指**：

- **CMC**：连接拇指与手掌的基关节
- **MCP**：掌指关节
- **PIP**：近端指间关节
- **DIP**：远端指间关节（指尖关节）

**食指、中指、无名指和小指**：

- **桡侧屈肌（Radial flexor）**：将手指根部向拇指方向移动
- **尺侧屈肌（Ulnar flexor）**：将手指根部向小指方向移动
- **PIP/DIP**：弯曲手指的远节和近节指骨

每个关节都需要通过图形界面单独校准。
请注意，尺侧屈肌和桡侧屈肌的范圍应大小相同（但偏移量不同），以实现对称运动。

### 1.2 校准遥操作数据手套

```bash
lerobot-calibrate \
    --teleop.type=homunculus_glove \
    --teleop.port=/dev/tty.usbmodem11201 \
    --teleop.id=red \
    --teleop.side=right
```

### 1.3 校准机器人机械臂

```bash
lerobot-calibrate \
    --robot.type=hope_jr_arm \
    --robot.port=/dev/tty.usbserial-1110 \
    --robot.id=white
```

这将打开一个校准图形界面，你可以在其中设置每个电机的范围限制。机械臂的运动组织如下：

- **肩部**：俯仰（pitch）、偏航（yaw）和翻滚（roll）
- **肘部**：屈曲（flex）
- **腕部**：俯仰（pitch）、偏航（yaw）和翻滚（roll）

## 第 2 步：遥操作

由于 Feetech 中间件中的全局变量冲突，机械臂和灵巧手的遥操作需要在不同的 shell 会话中运行：

### 灵巧手

```bash
lerobot-teleoperate \
    --robot.type=hope_jr_hand \
    --robot.port=/dev/tty.usbmodem58760432281 \
    --robot.id=blue \
    --robot.side=right \
    --teleop.type=homunculus_glove \
    --teleop.port=/dev/tty.usbmodem11201 \
    --teleop.id=red \
    --teleop.side=right \
    --display_data=true \
    --fps=30
```

### 机械臂

```bash
lerobot-teleoperate \
    --robot.type=hope_jr_arm \
    --robot.port=/dev/tty.usbserial-1110 \
    --robot.id=white \
    --teleop.type=homunculus_arm \
    --teleop.port=/dev/tty.usbmodem11201 \
    --teleop.id=black \
    --display_data=true \
    --fps=30
```

## 第 3 步：记录、回放、训练

Hope-JR 的记录、回放和训练功能仍处于实验阶段。

### 记录

此步骤记录数据集，可参考[此示例](https://huggingface.co/datasets/nepyope/hand_record_test_with_video_data/settings)。

```bash
lerobot-record \
    --robot.type=hope_jr_hand \
    --robot.port=/dev/tty.usbmodem58760432281 \
    --robot.id=right \
    --robot.side=right \
    --robot.cameras='{"main": {"type": "opencv", "index_or_path": 0, "width": 640, "height": 480, "fps": 30}}' \
    --teleop.type=homunculus_glove \
    --teleop.port=/dev/tty.usbmodem1201 \
    --teleop.id=right \
    --teleop.side=right \
    --dataset.repo_id=<USER>/hand_record_test_with_video_data \
    --dataset.single_task="手部记录测试（含视频数据）" \
    --dataset.num_episodes=1 \
    --dataset.episode_time_s=5 \
    --dataset.push_to_hub=true \
    --dataset.private=true \
    --dataset.streaming_encoding=true \
    --dataset.encoder_threads=2 \
    # --dataset.rgb_encoder.vcodec=auto \
    --display_data=true
```

### 回放

```bash
lerobot-replay \
    --robot.type=hope_jr_hand \
    --robot.port=/dev/tty.usbmodem58760432281 \
    --robot.id=right \
    --robot.side=right \
    --dataset.repo_id=<USER>/hand_record_test_with_camera \
    --dataset.episode=0
```

### 训练

```bash
lerobot-train \
  --dataset.repo_id=<USER>/hand_record_test_with_video_data \
  --policy.type=act \
  --output_dir=outputs/train/hopejr_hand \
  --job_name=hopejr \
  --policy.device=mps \
  --wandb.enable=true \
  --policy.repo_id=<USER>/hand_test_policy
```

### 评估

此训练运行可参考[此示例](https://wandb.ai/tino/lerobot/runs/rp0k8zvw?nw=nwusertino)。

```bash

```
