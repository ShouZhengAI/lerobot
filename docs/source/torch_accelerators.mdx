# PyTorch 加速器

LeRobot 支持多种硬件加速选项，适用于训练和推理。

这些选项包括：

- **CPU**：CPU 执行所有计算，不使用专用加速器
- **CUDA**：使用 NVIDIA 和 AMD GPU 进行加速
- **MPS**：使用 Apple Silicon GPU 进行加速
- **XPU**：使用 Intel 集成和独立 GPU 进行加速

## 入门

要使用特定的加速器，需要安装合适版本的 PyTorch。

对于 CPU、CUDA 和 MPS 后端，请遵循 [PyTorch 安装页面](https://pytorch.org/get-started/locally) 上的说明。
对于 XPU 后端，请遵循 [PyTorch 文档](https://docs.pytorch.org/docs/stable/notes/get_start_xpu.html) 中的说明。

### 验证安装

安装完成后，可以通过运行以下代码验证加速器是否可用：

```python
import torch
print(torch.<backend_name>.is_available())  # <backend_name> 可以是 cuda、mps 或 xpu
```

## 如何运行训练或评估

要选择所需的加速器，请在运行 `lerobot-train` 或 `lerobot-eval` 时使用 `--policy.device` 参数。例如，在 Apple Silicon 上使用 MPS，运行：

```bash
lerobot-train
    --policy.device=mps ...
```

```bash
lerobot-eval \
    --policy.device=mps ...
```

然而，在大多数情况下，加速器的存在会被自动检测到，`policy.device` 参数可以从 CLI 命令中省略。
