# Meta-World

Meta-World 是一个面向**多任务和元强化学习**（multi-task and meta reinforcement learning）的连续控制机器人操作开源模拟基准。它整合了 50 个使用日常物品和通用桌面 Sawyer 机械臂的多样化操作任务，为测试算法能否学习多个不同任务并快速泛化到新任务提供了标准化的实验平台。

- 论文：[Meta-World: A Benchmark and Evaluation for Multi-Task and Meta Reinforcement Learning paper](https://arxiv.org/abs/1910.10897)
- GitHub 仓库：[Farama-Foundation/Metaworld](https://github.com/Farama-Foundation/Metaworld)
- 项目网站：[metaworld.farama.org](https://metaworld.farama.org)

![MetaWorld MT10 demo](https://meta-world.github.io/figures/ml45.gif)

## 可用任务

Meta-World 提供 50 个任务，按难度分组。在 LeRobot 中，您可以评估单个任务、难度分组或完整的 MT50 任务套件：

| 分组       | CLI 名称              | 任务数量 | 描述                                |
| ---------- | -------------------- | ----- | ---------------------------------------- |
| 简单       | `easy`               | 28    | 具有简单动力学和单步目标的任务          |
| 中等       | `medium`             | 11    | 需要多步推理的任务                      |
| 困难       | `hard`               | 6     | 涉及复杂接触和精确操作的任务            |
| 极难       | `very_hard`          | 5     | 套件中最具挑战性的任务                  |
| MT50 (全部) | 逗号分隔列表         | 50    | 全部 50 个任务——最具挑战性的多任务设置  |

您也可以直接传入单个任务名称（例如 `assembly-v3`、`dial-turn-v3`）。

我们在 Hugging Face Hub 上提供了适用于 LeRobot 的 Meta-World MT50 数据集：[lerobot/metaworld_mt50](https://huggingface.co/datasets/lerobot/metaworld_mt50)。该数据集专为使用全部 50 个任务进行 MT50 评估而格式化，采用固定的物体/目标位置和 one-hot 任务向量以保证一致性。

## 安装

在完成 LeRobot 安装步骤后：

```bash
pip install -e ".[metaworld]"
```

<Tip warning={true}>
如果在运行 Meta-World 环境时遇到 `AssertionError: ['human', 'rgb_array', 'depth_array']`，这是 Meta-World 与您的 Gymnasium 版本不匹配。请按以下方式修复：

```bash
pip install "gymnasium==1.1.0"
```

</Tip>

## 评估

### 默认评估（推荐）

在中等难度分组上评估（兼顾覆盖面和计算开销的较好选择）：

```bash
lerobot-eval \
  --policy.path="your-policy-id" \
  --env.type=metaworld \
  --env.task=medium \
  --eval.batch_size=1 \
  --eval.n_episodes=10
```

### 单任务评估

评估指定任务：

```bash
lerobot-eval \
  --policy.path="your-policy-id" \
  --env.type=metaworld \
  --env.task=assembly-v3 \
  --eval.batch_size=1 \
  --eval.n_episodes=10
```

### 多任务评估

跨多个任务或难度分组进行评估：

```bash
lerobot-eval \
  --policy.path="your-policy-id" \
  --env.type=metaworld \
  --env.task=assembly-v3,dial-turn-v3,handle-press-side-v3 \
  --eval.batch_size=1 \
  --eval.n_episodes=10
```

- `--env.task` 接受明确的任务列表（逗号分隔）或难度分组名称（例如 `easy`、`medium`、`hard`、`very_hard`）。
- `--eval.batch_size` 控制同时运行多少个环境。
- `--eval.n_episodes` 设置每个任务运行的回合数。

### 策略输入与输出

**观测（Observations）：**

- `observation.image` —— 单摄像头视角（`corner2`），480x480 HWC uint8
- `observation.state` —— 4 维本体感受状态（末端执行器位置 + 夹爪）

**动作（Actions）：**

- `Box(-1, 1, shape=(4,))` 中的连续控制 —— 3D 末端执行器增量 + 1D 夹爪

### 推荐评估回合数

为获得可复现的基准测试结果，建议每个任务运行 **10 个回合**。对于完整的 MT50 套件，总计 500 个回合。如果您关心泛化能力，请在完整 MT50 上运行评估——它本身具有挑战性，能比少数窄任务更好地揭示模型的能力强弱。

## 训练

### 训练命令示例

在 Meta-World 任务子集上训练 SmolVLA 策略：

```bash
lerobot-train \
  --policy.type=smolvla \
  --policy.repo_id=${HF_USER}/metaworld-test \
  --policy.load_vlm_weights=true \
  --dataset.repo_id=lerobot/metaworld_mt50 \
  --env.type=metaworld \
  --env.task=assembly-v3,dial-turn-v3,handle-press-side-v3 \
  --output_dir=./outputs/ \
  --steps=100000 \
  --batch_size=4 \
  --eval.batch_size=1 \
  --eval.n_episodes=1 \
  --env_eval_freq=1000
```

## 实用建议

- 在多任务训练中使用 one-hot 任务条件化（按 MT10/MT50 约定），使策略获得明确的任务上下文。
- 编写后处理或日志记录时，检查数据集的任务描述和 `info["is_success"]` 键，确保您的成功率指标与基准对齐。
- 根据您的计算预算调整 `batch_size`、`steps` 和 `env_eval_freq`。
