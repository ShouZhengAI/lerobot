# OpenArm

[OpenArm](https://openarm.dev) 是一款开源的 7 自由度（7DOF）人形机械臂，专为实体 AI（Physical AI）研究和部署而设计。

如需获取 OpenArm（组装件或 DIY 套件）并加入全球社区，请访问 [openarm.dev](https://openarm.dev) 浏览全球认证制造商列表。

## 独特之处

- **拟人尺寸设计**：OpenArm 采用类人比例设计，适配身高约 160-165cm 的人体尺寸。在安全灵敏的操作中，兼顾了实际工作范围与可控惯性的最佳平衡。

- **安全优先架构**：采用 QDD 反向驱动电机和高顺服性设计，OpenArm 在保证安全人机交互的同时，具备可满足实际任务的负载能力（峰值 6.0kg / 额定 4.1kg）。

- **耐久性设计**：关键结构部件采用铝合金和不锈钢制造，确保在重复数据采集和持续研究使用中的稳定性能。

- **完全开放可搭建**：从 CNC 零件和 3D 打印外壳到电气布线，每个组件均可由研究人员和实验室自行采购和组装，并提供完整的制造数据。

- **实用且经济**：完整双臂系统的价格为 6,500 美元，OpenArm 以传统人形机器人成本的一小部分，提供研究级性能。

## 平台要求

<Tip warning={true}>
  **仅支持 Linux**：OpenArm 当前仅支持 Linux 系统。CAN 总线 USB 适配器
  没有 macOS 驱动，尚未在 Windows 上测试。

## 安全指南

操作 OpenArm 前，请阅读[官方安全指南](https://docs.openarm.dev/getting-started/safety-guide)。要点如下：

- **牢固安装**：用螺丝或夹具将机械臂固定在平坦稳定的表面上
- **安全距离**：操作期间确保身体部位和物体处于运动范围之外
- **防护装备**：始终佩戴安全护目镜；根据需要佩戴其他个人防护装备（PPE）
- **负载限制**：请勿超过规定的负载限制（单臂峰值 6.0kg / 额定 4.1kg）
- **急停装置**：了解急停装置的位置和操作方法
- **定期检查**：检查螺丝是否松动、机械限位是否损坏、是否有异常噪音以及线路是否损坏

## CAN 总线设置

OpenArm 使用 CAN 总线与 Damiao 电机通信。将 CAN 总线 USB 适配器连接到 Linux 电脑后，请按照 [Damiao 电机和 CAN 总线指南](./damiao) 配置接口。

快速配置：

```bash
# 设置 CAN 接口
lerobot-setup-can --mode=setup --interfaces=can0,can1

# 测试电机通信
lerobot-setup-can --mode=test --interfaces=can0,can1
```

## 使用方法

### 从动臂（机器人端）

<hfoptions id="follower">
<hfoption id="命令行">

```bash
lerobot-calibrate \
    --robot.type=openarm_follower \
    --robot.port=can0 \
    --robot.side=right \
    --robot.id=my_openarm_follower
```

</hfoption>
<hfoption id="API 示例">

```python
from lerobot.robots.openarm_follower import OpenArmFollower, OpenArmFollowerConfig

config = OpenArmFollowerConfig(
    port="can0",
    side="right",  # 或使用 "left" 表示左臂
    id="my_openarm_follower",
)

follower = OpenArmFollower(config)
follower.connect()

# 读取当前状态
obs = follower.get_observation()
print(obs)

# 发送动作（角度制位置）
action = {
    "joint_1.pos": 0.0,
    "joint_2.pos": 0.0,
    "joint_3.pos": 0.0,
    "joint_4.pos": 45.0,
    "joint_5.pos": 0.0,
    "joint_6.pos": 0.0,
    "joint_7.pos": 0.0,
    "gripper.pos": 0.0,
}
follower.send_action(action)

follower.disconnect()
```

</hfoption>
</hfoptions>

### 主动臂（遥操作端）

主动臂用于遥操作——通过手动移动来控制从动臂。

<hfoptions id="leader">
<hfoption id="命令行">

```bash
lerobot-calibrate \
    --teleop.type=openarm_leader \
    --teleop.port=can1 \
    --teleop.id=my_openarm_leader
```

</hfoption>
<hfoption id="API 示例">

```python
from lerobot.teleoperators.openarm_leader import OpenArmLeader, OpenArmLeaderConfig

config = OpenArmLeaderConfig(
    port="can1",
    id="my_openarm_leader",
    manual_control=True,  # 关闭力矩输出以进行手动移动
)

leader = OpenArmLeader(config)
leader.connect()

# 读取当前位置（作为发送给从动臂的动作）
action = leader.get_action()
print(action)

leader.disconnect()
```

</hfoption>
</hfoptions>

### 遥操作

使用主从控制遥操作 OpenArm：

```bash
lerobot-teleoperate \
    --robot.type=openarm_follower \
    --robot.port=can0 \
    --robot.side=right \
    --robot.id=my_follower \
    --teleop.type=openarm_leader \
    --teleop.port=can1 \
    --teleop.id=my_leader
```

### 双臂遥操作

使用两个主动臂和两个从动臂进行双臂遥操作设置：

```bash
lerobot-teleoperate \
    --robot.type=bi_openarm_follower \
    --robot.left_arm_config.port=can0 \
    --robot.left_arm_config.side=left \
    --robot.right_arm_config.port=can1 \
    --robot.right_arm_config.side=right \
    --robot.id=my_bimanual_follower \
    --teleop.type=bi_openarm_leader \
    --teleop.left_arm_config.port=can2 \
    --teleop.right_arm_config.port=can3 \
    --teleop.id=my_bimanual_leader
```

### 录制数据

在遥操作过程中录制数据集：

```bash
lerobot-record \
    --robot.type=openarm_follower \
    --robot.port=can0 \
    --robot.side=right \
    --robot.id=my_follower \
    --teleop.type=openarm_leader \
    --teleop.port=can1 \
    --teleop.id=my_leader \
    --repo-id=my_hf_username/my_openarm_dataset \
    --fps=30 \
    --num-episodes=10
```

## 配置选项

### 从动臂配置

| 参数                 | 默认值     | 说明                                            |
| --------------------- | --------- | ----------------------------------------------- |
| `port`                | -         | CAN 接口（例如 `can0`）                          |
| `side`                | `None`    | 机械臂侧：`"left"`、`"right"`，或自定义限位用 `None` |
| `use_can_fd`          | `True`    | 启用 CAN FD 以获得更高数据传输速率                 |
| `can_bitrate`         | `1000000` | 标称比特率（1 Mbps）                              |
| `can_data_bitrate`    | `5000000` | CAN FD 数据比特率（5 Mbps）                        |
| `max_relative_target` | `None`    | 相对目标位置的安全限制                            |
| `position_kp`         | 逐关节     | 位置控制比例增益                                  |
| `position_kd`         | 逐关节     | 位置控制微分增益                                  |

### 主动臂配置

| 参数             | 默认值     | 说明                              |
| ----------------- | --------- | --------------------------------- |
| `port`            | -         | CAN 接口（例如 `can1`）            |
| `manual_control`  | `True`    | 关闭力矩输出以进行手动移动          |
| `use_can_fd`      | `True`    | 启用 CAN FD 以获得更高数据传输速率  |
| `can_bitrate`     | `1000000` | 标称比特率（1 Mbps）               |
| `can_data_bitrate`| `5000000` | CAN FD 数据比特率（5 Mbps）         |

## 电机配置

OpenArm 使用 Damiao 电机，默认配置如下：

| 关节                          | 电机型号 | 发送 ID | 接收 ID |
| ----------------------------- | ------- | ------- | ------- |
| joint_1（肩部水平旋转）       | DM8009  | 0x01    | 0x11    |
| joint_2（肩部抬升）           | DM8009  | 0x02    | 0x12    |
| joint_3（肩部旋转）           | DM4340  | 0x03    | 0x13    |
| joint_4（肘部弯曲）           | DM4340  | 0x04    | 0x14    |
| joint_5（腕部旋转）           | DM4310  | 0x05    | 0x15    |
| joint_6（腕部俯仰）           | DM4310  | 0x06    | 0x16    |
| joint_7（腕部回转）           | DM4310  | 0x07    | 0x17    |
| 夹爪                           | DM4310  | 0x08    | 0x18    |

## 故障排除

### 电机无响应

1. 检查电源连接
2. 确认 CAN 接线（CAN-H、CAN-L、GND）
3. 运行诊断：`lerobot-setup-can --mode=test --interfaces=can0`
4. 更多详情请参阅 [Damiao 故障排除指南](./damiao#troubleshooting)

### 找不到 CAN 接口

确保 CAN 接口已配置：

```bash
ip link show can0
```
