# ACT（基于Transformer的动作分块）

ACT 是一种**轻量且高效的模仿学习策略**，特别适用于精细操作任务。它是我们**初次使用 LeRobot 时推荐的首选模型**，因为其训练速度快、计算需求低且性能优异。

## 模型概述

Action Chunking with Transformers (ACT，基于Transformer的动作分块) 由 Zhao 等人在论文 [Learning Fine-Grained Bimanual Manipulation with Low-Cost Hardware](https://arxiv.org/abs/2304.13705) 中提出。该策略旨在利用低成本硬件和最少的示教数据完成精确、高接触的操控任务。

### ACT 为何适合初学者

ACT 作为入门首选有以下几个突出优点：

- **训练速度快**：在单张 GPU 上仅需数小时即可完成训练
- **轻量级**：仅约 8000 万个参数，高效且易于使用
- **数据高效**：通常仅需 50 条示教数据即可达到较高成功率

### 架构

ACT 采用基于 Transformer 的架构，包含三个主要组件：

1. **视觉骨干网络（Vision Backbone）**：ResNet-18 处理来自多个相机视角的图像
2. **Transformer 编码器（Encoder）**：融合相机特征、关节位置以及一个可学习的隐变量
3. **Transformer 解码器（Decoder）**：通过交叉注意力（cross-attention）生成连贯的动作序列

该策略的输入包括：

- 多张 RGB 图像（例如来自腕部相机、前置/顶部相机）
- 当前机器人关节位置
- 隐变量 `z`（训练过程中学习，推理时设为零）

输出为包含 `k` 个未来动作序列的动作块（action chunk）。

## 安装要求

1. 按照我们的[安装指南](./installation)安装 LeRobot
2. ACT 已包含在 LeRobot 基础安装中，无需额外依赖！

## 训练 ACT

ACT 与标准的 LeRobot 训练流程完全兼容。以下是在你的数据集上训练 ACT 的完整示例：

```bash
lerobot-train \
  --dataset.repo_id=${HF_USER}/your_dataset \
  --policy.type=act \
  --output_dir=outputs/train/act_your_dataset \
  --job_name=act_your_dataset \
  --policy.device=cuda \
  --wandb.enable=true \
  --policy.repo_id=${HF_USER}/act_policy
```

### 训练建议

1. **从默认参数开始**：ACT 的默认超参数在大多数任务上表现良好
2. **训练时长**：在单张 GPU 上运行 10 万步训练步骤约需数小时
3. **批大小（batch size）**：从批大小 8 开始，根据你的 GPU 内存进行调整

### 使用 Google Colab 训练

如果你的本地计算机没有强大的 GPU，可以按照[ACT 训练笔记本](./notebooks#training-act)使用 Google Colab 训练模型。

## 评估 ACT

训练完成后，你可以使用 `lerobot-record` 命令配合训练好的策略来评估 ACT。该命令将运行推理并记录评估片段：

```bash
lerobot-rollout \
  --strategy.type=base \
  --policy.path=${HF_USER}/act_policy \
  --robot.type=so101_follower \
  --robot.port=/dev/ttyACM0 \
  --robot.cameras="{ front: {type: opencv, index_or_path: 0, width: 640, height: 480, fps: 30}}" \
  --display_data=true \
  --task="Your task description" \ # ACT 可省略此参数
  --duration=60
```
