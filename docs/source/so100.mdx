# SO-100

以下步骤说明如何组装 SO-100 机器人。

## 获取零件

请参考此 [README](https://github.com/TheRobotStudio/SO-ARM100/blob/main/SO100.md) 文档。其中包含物料清单、零件采购链接以及 3D 打印零件的说明。如果你是第一次打印或没有 3D 打印机，文档也提供了相关建议。

## 安装 LeRobot 🤗

安装 LeRobot，请按照我们的[安装指南](./installation)进行操作。

除上述说明外，你还需要安装 Feetech SDK：

```bash
pip install -e ".[feetech]"
```

## 配置电机

**注意：**
与 SO-101 不同，SO-100 的电机连接器在手臂组装完成后不易接触，因此配置步骤必须提前完成。

### 1. 找到每个手臂对应的 USB 端口

要找到每个总线舵机适配器的端口，请运行以下脚本：

```bash
lerobot-find-port
```

<hfoptions id="example">
<hfoption id="Mac">

示例输出：

```
Finding all available ports for the MotorBus.
['/dev/tty.usbmodem575E0032081', '/dev/tty.usbmodem575E0031751']
Remove the USB cable from your MotorsBus and press Enter when done.

[...断开对应的主臂或从臂连接，然后按 Enter...]

The port of this MotorsBus is /dev/tty.usbmodem575E0032081
Reconnect the USB cable.
```

找到的端口为：`/dev/tty.usbmodem575E0032081`，对应你的主臂或从臂。

</hfoption>

```bash

sudo chmod 666 /dev/ttyACM1
```

示例输出：

```

['/dev/ttyACM0', '/dev/ttyACM1']

[...断开对应的主臂或从臂连接，然后按 Enter...]

The port of this MotorsBus is /dev/ttyACM1
Reconnect the USB cable.
```

找到的端口为：`/dev/ttyACM1`，对应你的主臂或从臂。

### 2. 设置电机 ID 和波特率

每个电机在总线上通过唯一的 ID 进行标识。新电机出厂时通常默认 ID 为 `1`。为了使电机与控制器之间的通信正常工作，我们首先需要为每个电机设置不同的唯一 ID。此外，数据的传输速度由波特率决定。为了实现通信，控制器和所有电机需要配置相同的波特率。

为此，我们需要先使用控制器单独连接每个电机来设置这些参数。由于我们会将这些参数写入电机内存的非易失性区域（EEPROM），因此只需执行一次。

如果你是从其他机器人上重新利用电机，很可能也需要执行此步骤，因为原有的 ID 和波特率很可能不匹配。

#### 从臂

将计算机的 USB 线和电源连接到从臂的控制板。然后，运行以下命令或使用上一步得到的端口运行 API 示例。你还需要使用 `id` 参数为主臂命名。

关于如何设置电机 ID 的视觉参考，请参阅[此视频](https://huggingface.co/docs/lerobot/en/so101#setup-motors-video)，其中我们演示了 SO101 手臂的设置过程。

<hfoptions id="setup_motors">
<hfoption id="命令">

```bash
lerobot-setup-motors \
    --robot.type=so100_follower \
    --robot.port=/dev/tty.usbmodem585A0076841  # <- 在此粘贴上一步找到的端口
```

</hfoption>
<hfoption id="API 示例">

<!-- prettier-ignore-start -->
```python
from lerobot.robots.so_follower import SO100Follower, SO100FollowerConfig

config = SO100FollowerConfig(
    port="/dev/tty.usbmodem585A0076841",
    id="my_awesome_follower_arm",
)
follower = SO100Follower(config)
follower.setup_motors()
```
<!-- prettier-ignore-end -->

</hfoption>
</hfoptions>

你应该会看到以下提示：

```
Connect the controller board to the 'gripper' motor only and press enter.
```

按照提示，仅插入夹爪电机。确保该电机是唯一连接到控制板的电机，并且电机本身尚未与其他任何电机菊花链连接。当你按下 `[Enter]` 后，脚本将自动设置该电机的 ID 和波特率。

<details>
<summary>故障排除</summary>

如果此时出现错误，请检查线缆是否连接正确：

<ul>
  <li>电源</li>
  <li>计算机与控制板之间的 USB 线</li>
  <li>从控制板到电机的 3 针线缆</li>
</ul>

如果你使用的是 Waveshare 控制板，请确保两个跳线都设置在 `B` 通道（USB）上。

</details>

然后你应该会看到以下消息：

```
'gripper' motor id set to 6
```

接着是下一步提示：

```
Connect the controller board to the 'wrist_roll' motor only and press enter.
```

你可以将 3 针线缆从控制板上断开，但可以将其另一端留在夹爪电机上，因为它已经安装在正确位置。现在，将另一根 3 针线缆插入腕部旋转电机并连接到控制板。与上一个电机一样，确保它是唯一连接到控制板的电机，并且该电机本身未连接任何其他电机。

按提示为每个电机重复此操作。

> [!TIP]
> 在每个步骤按 Enter 之前检查一下线缆。例如，在操作控制板时，电源线可能会意外断开。

完成后，脚本将自动结束，此时电机即可正常使用。现在，你可以将每个电机的 3 针线缆依次连接到下一个电机，并将第一个电机（ID=1 的'肩部旋转'电机）的线缆连接到控制板，然后将控制板安装到手臂底座。

## 校准

接下来，你需要校准机器人，以确保主臂和从臂在相同物理位置时具有相同的位置值。
校准过程非常重要，因为它使在一个机器人上训练的神经网络能够在另一个机器人上工作。

#### 从臂

运行以下命令或 API 示例来校准从臂：

<hfoptions id="calibrate_follower">
<hfoption id="命令">

```bash
lerobot-calibrate \
    --robot.type=so100_follower \
    --robot.port=/dev/tty.usbmodem58760431551 \ # <- 机器人的端口
    --robot.id=my_awesome_follower_arm # <- 为机器人指定一个唯一名称
```

</hfoption>
<hfoption id="API 示例">

<!-- prettier-ignore-start -->
```python
from lerobot.robots.so_follower import SO100FollowerConfig, SO100Follower

config = SO100FollowerConfig(
    port="/dev/tty.usbmodem585A0076891",
    id="my_awesome_follower_arm",
)

follower = SO100Follower(config)
follower.connect(calibrate=False)
follower.calibrate()
follower.disconnect()
```
<!-- prettier-ignore-end -->

</hfoption>
</hfoptions>

我们对大多数机器人的校准方法进行了统一。因此，SO100 手臂的校准步骤与 Koch 和 SO101 相同。首先，我们需要将机器人移动到每个关节处于其运动范围中间的位置，然后按 `Enter`。其次，将所有关节移动通过其完整的运动范围。关于 SO101 相同过程的视频参考，请点击[此处](https://huggingface.co/docs/lerobot/en/so101#calibration-video)。

#### 主臂

执行相同步骤校准主臂，运行以下命令或 API 示例：

<hfoptions id="calibrate_leader">
<hfoption id="命令">

