# reBot B601-DM

[reBot B601-DM](https://wiki.seeedstudio.com/rebot_arm_b601_dm_lerobot/) 是 Seeed Studio 出品的一款开源、低成本的机器人臂，适用于具身人工智能和模仿学习。它包含一个**从动臂**（`B601-DM`，由 Damiao CAN 电机驱动的 6 自由度臂加夹爪）和一个**主控臂**（`StarArm102` / `reBot Arm 102`，由 FashionStar UART 智能舵机驱动），后者用于遥操作前者。

本页面涵盖**标定**和**遥操作**，包括单臂和双臂两种设置。

## 安装 LeRobot

按照我们的[安装指南](./installation)操作，然后安装 reBot 支持：

```bash
pip install -e ".[rebot]"
```

这会安装 `motorbridge`（用于 B601-DM 从动臂的 CAN 电机控制）和 `motorbridge-smart-servo`（用于 reBot Arm 102 主控臂的 FashionStar UART 舵机）。

## 已注册的设备类型

| 类型                      | 说明                           |
| ------------------------- | ------------------------------ |
| `rebot_b601_follower`     | 单臂 B601-DM 从动机器人          |
| `bi_rebot_b601_follower`  | 双臂从动机器人                   |
| `rebot_102_leader`        | 单臂 reBot Arm 102 主控遥操作设备 |
| `bi_rebot_102_leader`     | 双臂主控遥操作设备               |

双臂类型组合了两个单臂实例，并在每个臂的观测/动作键前加上 `left_` / `right_` 前缀。每个臂的设置通过嵌套的 `left_arm_config.*` / `right_arm_config.*` 参数传递。

## 查找 USB 端口

对于每个设备，使用以下命令查找与其电机总线关联的 USB 端口：

```bash
lerobot-find-port
```

<Tip warning={true}>
  在 Linux 上，请移除 `brltty`（`sudo apt remove brltty`），以免它占用主控臂的 USB 串口。您可能还需要授予串行设备的访问权限：`sudo chmod 666 /dev/ttyACM* /dev/ttyUSB*`。
</Tip>

## 标定

两个臂都不存储持久的硬件标定：每次连接时，电机会根据臂当前的实际姿态重新归零。标定仅记录这个零位姿态。在提示时，**手动将臂移动到其零位**（上面所示的默认坐姿，夹爪完全闭合），然后按 <kbd>ENTER</kbd> 键。

### 从动臂（B601-DM）

<hfoptions id="calibrate-follower">
<hfoption id="单臂">

```bash
lerobot-calibrate \
    --robot.type=rebot_b601_follower \
    --robot.port=/dev/ttyACM0 \
    --robot.id=follower \
    --robot.can_adapter=damiao
```

</hfoption>
<hfoption id="双臂">

连接双臂从动机器人；标定将依次运行左臂和右臂。

```bash
lerobot-calibrate \
    --robot.type=bi_rebot_b601_follower \
    --robot.id=bi_follower \
    --robot.left_arm_config.port=/dev/ttyACM0 \
    --robot.left_arm_config.can_adapter=damiao \
    --robot.right_arm_config.port=/dev/ttyACM1 \
    --robot.right_arm_config.can_adapter=damiao
```

每个臂的标定文件将以 `_left` / `_right` 后缀保存在 id 上。

</hfoption>
</hfoptions>

### 主控臂（reBot Arm 102）

<hfoptions id="calibrate-leader">
<hfoption id="单臂">

```bash
lerobot-calibrate \
    --teleop.type=rebot_102_leader \
    --teleop.port=/dev/ttyUSB0 \
    --teleop.id=leader
```

</hfoption>
<hfoption id="双臂">

```bash
lerobot-calibrate \
    --teleop.type=bi_rebot_102_leader \
    --teleop.id=bi_leader \
    --teleop.left_arm_config.port=/dev/ttyUSB0 \
    --teleop.right_arm_config.port=/dev/ttyUSB1
```

</hfoption>
</hfoptions>

## 遥操作

两个臂都标定完成后，用主控臂驱动从动臂。从动臂通过 Damiao 串行桥（`can_adapter=damiao`，默认值）或 SocketCAN 适配器（`can_adapter=socketcan`）与其 CAN 总线通信。有关 SocketCAN 适配器配置的更多详情，请参见 [OpenArm 页面](./openarm)。

<hfoptions id="teleoperate">
<hfoption id="单臂">

```bash
lerobot-teleoperate \
    --robot.type=rebot_b601_follower \
    --robot.port=/dev/ttyACM0 \
    --robot.id=follower \
    --robot.can_adapter=damiao \
    --teleop.type=rebot_102_leader \
    --teleop.port=/dev/ttyUSB0 \
    --teleop.id=leader
```

</hfoption>
<hfoption id="双臂">

双臂的主控臂和从动臂复用单臂类；每个臂通过嵌套的 `left_arm_config.*` / `right_arm_config.*` 参数进行配置，因此双臂的 reBot Arm 102 主控臂可以驱动双臂的 B601-DM 从动臂。

```bash
lerobot-teleoperate \
    --robot.type=bi_rebot_b601_follower \
    --robot.id=bi_follower \
    --robot.left_arm_config.port=/dev/ttyACM0 \
    --robot.left_arm_config.can_adapter=damiao \
    --robot.right_arm_config.port=/dev/ttyACM1 \
    --robot.right_arm_config.can_adapter=damiao \
    --teleop.type=bi_rebot_102_leader \
    --teleop.id=bi_leader \
    --teleop.left_arm_config.port=/dev/ttyUSB0 \
    --teleop.right_arm_config.port=/dev/ttyUSB1
```

</hfoption>
</hfoptions>

<Tip>
  主控臂和从动臂共享相同的关节名称（`shoulder_pan`（肩部回转）、`shoulder_lift`（肩部抬升）、`elbow_flex`（肘部弯曲）、`wrist_flex`（腕部弯曲）、`wrist_yaw`（腕部偏航）、`wrist_roll`（腕部翻滚）、`gripper`（夹爪）），因此主控臂的动作可以直接映射到从动臂上。
</Tip>

如果某个关节的运动方向相反，可以在主控臂的 `joint_directions` 中翻转其符号（夹爪还有一个缩放因子，用于将范围扩展到从动臂的夹爪范围）：

```bash
lerobot-teleoperate \
    --robot.type=rebot_b601_follower \
    --robot.port=/dev/ttyACM0 \
    --robot.can_adapter=damiao \
    --teleop.type=rebot_102_leader \
    --teleop.port=/dev/ttyUSB0 \
    --teleop.joint_directions='{"shoulder_pan":-1,"shoulder_lift":-1,"elbow_flex":1,"wrist_flex":1,"wrist_yaw":1,"wrist_roll":-1,"gripper":-6}'
```

## 录制数据集

将 `lerobot-teleoperate` 替换为 `lerobot-record`（使用相同的 `--robot.*` / `--teleop.*` 参数，再加上 `--dataset.*`）来录制训练用的演示数据。完整工作流程请参见[机器人模仿学习](./il_robots)。

有关硬件组装和接线，请参见 [Seeed Studio reBot 维基页面](https://wiki.seeedstudio.com/rebot_arm_b601_dm_lerobot/)。
